A Header

You put a set of rules in a <style> tag, and they are applied to parts of the documents that match the selectors.

After a while we end up with far too many CSS rules to put in the <head> area of each document. A simpler way to include a large number of CSS rules is to put them in a separate file and ⇒ include that file in each of the pages.
